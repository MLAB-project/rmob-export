rmob-export
===========

Generator of colorgramme graphs to rmob observers database.

### Ubuntu 13.04

sudo apt-get install libmagick++4
sudo apt-get install libmagick++-dev
 

## Howto

### Compiling

1. cd to project in rmob-export/ directory

2. make


### Running

./RmobGen -input-[Path-to-.info-file]
